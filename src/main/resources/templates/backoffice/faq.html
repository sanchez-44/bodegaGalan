Guia del Usuario - Bodega Galan

1. Registrar Venta

Esta opción te llevará a una página donde puedes registrar una nueva venta. Podrás ingresar detalles como el cliente, los productos vendidos, la cantidad, etc. La información ingresada se almacenará en la base de datos para su posterior procesamiento.

2. Registro de Ventas

Al seleccionar esta opción, accederás a una página que muestra un historial o lista de todas las ventas realizadas. Puedes ver información detallada sobre cada venta, como fecha, cliente, productos comprados, etc.

3. Comprobante de Pago

Al hacer clic en esta opción, podrás generar un comprobante de pago para una venta específica. El comprobante puede incluir detalles como el total de la venta, los productos comprados y otros detalles relevantes.

4. FAQ (Preguntas Frecuentes)

Acceder a esta sección te permitirá encontrar respuestas a preguntas frecuentes sobre el funcionamiento de la tienda en línea. Las preguntas y respuestas suelen abordar temas como el proceso de compra, políticas de devolución, etc.

5. Registro de Usuario

Al seleccionar esta opción, podrás acceder a un formulario para registrar nuevos usuarios. Los usuarios pueden proporcionar información como nombre, dirección, correo electrónico, etc., para crear una cuenta en la tienda.

6. Registrar Proveedores

Esta opción te llevará a una página donde puedes registrar nuevos proveedores para los productos de la tienda. Puedes ingresar detalles como el nombre del proveedor, información de contacto, etc.

7. Registro de Productos

Al seleccionar esta opción, podrás agregar nuevos productos al inventario de la tienda. Puedes ingresar detalles como nombre del producto, precio, cantidad disponible, etc.

8. Nosotros

Al hacer clic en esta opción, podrás acceder a información sobre la empresa, su historia, valores y cualquier otra información relevante.

9. Cerrar Sesión

Al seleccionar esta opción, se cerrará la sesión actual del usuario.
